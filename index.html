BikeNet Web Bluetooth

Multi-step connect flow to discover the hub MAC and unlock actions.

1. Connection

Click Connect to pick the eLink hub from the Web Bluetooth picker.

No hub was selected. If the picker did not appear, make sure Web Bluetooth is enabled and try again.

2. MAC acquisition

We need the hub MAC to send commands. Use one of the methods below.

Auto discover (ads)

We listen for manufacturer data in advertisements.

Listening for advertisements...

Keep the hub awake and nearby. This is the fastest path.

Shift a gear

Shift once so we receive a Shift Complete event with the MAC.

Waiting for a shift-complete notification...

A single shift up or down is enough.

Manual entry

Find the MAC with nRF Connect or any Bluetooth scanner app.

3. Actions

MAC locked in. You can query and control the hub.

Log

Notifications and command results appear here.
