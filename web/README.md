# OpenElin

This repository primarily contains the **OpenElin web app**: a browser UI to configure the NXS shifter directly from your browser.

## Web app (primary)

Quick start:

```bash
npm install
npm run dev
```

Open:

- https://localhost:5173

## Node app (legacy)

There is also a Node BLE scanner entrypoint (`npm start`, `noble`-based).

For current development work, **ignore it completely**. The active product surface is the web app.
